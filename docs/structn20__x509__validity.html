char const* n20_x509_validity::not_after

The certificate shall not be valid after.

Must be initialize to a generalized time string of the form as described in N20_ASN1_TAG_UTC_TIME, or NULL. If NULL, the not after field of the certificate will be set to n20_x509_no_expiration.

◆ not_before

char const* n20_x509_validity::not_before

The certificate shall not be valid before.

Must be initialize to a generalized time string of the form as described in N20_ASN1_TAG_UTC_TIME, or NULL. If NULL, the not before field of the certificate will be set to n20_x509_unix_epoch.
